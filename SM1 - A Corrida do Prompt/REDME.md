# 🏎️ SM1 - A Corrida do Prompt

## 📌 Sobre o Projeto

O **SM1 - A Corrida do Prompt** é um projeto focado em engenharia de prompts e interação com modelos de Inteligência Artificial.  
O objetivo principal é explorar técnicas de criação, otimização e comparação de prompts para obter respostas mais eficientes, precisas e contextualizadas em modelos generativos.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Inteligência Artificial Generativa
- Engenharia de Prompt

---

## 🎯 Objetivos

- Aprender fundamentos de Prompt Engineering
- Testar diferentes estruturas de prompts
- Comparar qualidade de respostas
- Melhorar contexto e precisão das interações com IA

---

## 🧠 Conceitos Trabalhados

- Zero-shot Prompting
- Few-shot Prompting
- Chain of Thought
- Context Window
- Estruturação de Inputs
- Refinamento Iterativo
