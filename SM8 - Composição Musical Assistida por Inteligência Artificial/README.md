## 📝 Descrição do Projeto

Este projeto explora a integração entre criatividade humana e Inteligência Artificial aplicada à música. Utilizando técnicas de **RAG (Retrieval-Augmented Generation)** através do **Gemini Gems**, foi desenvolvido um assistente especializado na análise lírica e geração criativa de composições musicais inspiradas no gênero sertanejo e piseiro.

O principal objetivo foi criar um **EP com 5 faixas inéditas**, onde as letras foram produzidas com apoio da IA e a geração musical realizada através do **Suno AI**, permitindo a criação completa de melodias, vocais e instrumentais de forma automatizada.

---

## 🚀 Objetivos do Projeto

- Explorar o uso de IA na composição musical
- Aplicar técnicas de Prompt Engineering
- Utilizar RAG para personalização de letras
- Gerar músicas completas com IA generativa
- Documentar o processo criativo utilizando GitHub

---

## 🛠️ Tecnologias Utilizadas

- **Gemini Gems** → Criação de base de conhecimento e geração de letras
- **Suno AI** → Produção musical e geração de áudio
- **Git & GitHub** → Versionamento e documentação do projeto
