## 📌 Sobre o Projeto

O **SM4 — Engenharia Reversa** é um projeto de estudo focado em **análise estrutural de sistemas**, com o objetivo de compreender como aplicações funcionam internamente a partir da observação de seu comportamento, lógica e arquitetura.

A proposta é investigar, decompor e reconstruir funcionalidades para aprofundar o entendimento de **fluxos de software e design de sistemas**.

---

# 🧪 Visão do Processo

```text
Sistema existente → Análise estrutural → Mapeamento de fluxo → Recriação funcional
```
