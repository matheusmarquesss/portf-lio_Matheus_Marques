## 📌 Sobre o Projeto

O **SM6 — Engenharia de Software e IA com Bubble.io** é um projeto focado em desenvolvimento **no-code**, utilizando a plataforma Bubble.io para criação de aplicações rápidas com integração de Inteligência Artificial.

O objetivo é demonstrar como é possível construir sistemas inteligentes sem programação tradicional, usando **workflows visuais + APIs + IA generativa**.

---

# 🧠 Visão Geral da Arquitetura

```text
Usuário → Interface Bubble → Workflows → APIs de IA → Resposta Inteligente
```
