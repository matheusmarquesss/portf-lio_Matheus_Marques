# 🧪 SM2 - Laboratório de Classificação Visual

## 📌 Sobre o Projeto

O projeto **Laboratório de Classificação Visual** tem como objetivo explorar conceitos de visão computacional e classificação de imagens utilizando Inteligência Artificial.

A aplicação realiza análises visuais e classificação de elementos com base em modelos treinados ou APIs de reconhecimento visual.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- APIs de Visão Computacional
- Inteligência Artificial

---

## 🎯 Funcionalidades

- Upload de imagens
- Classificação visual automática
- Identificação de objetos
- Exibição de resultados em tempo real

---

## 🧠 Conceitos Trabalhados

- Computer Vision
- Image Classification
- Machine Learning
- Reconhecimento de padrões
- Processamento de imagem
