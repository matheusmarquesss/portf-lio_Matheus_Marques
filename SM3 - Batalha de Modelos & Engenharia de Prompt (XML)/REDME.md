# ⚔️ SM3 - Batalha de Modelos & Engenharia de Prompt (XML)

## 📌 Sobre o Projeto

Este projeto compara diferentes modelos de Inteligência Artificial utilizando técnicas avançadas de engenharia de prompt com estruturação em XML.

O foco está na padronização de entradas para melhorar previsibilidade, contexto e qualidade das respostas geradas pelos modelos.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- XML Prompting
- Inteligência Artificial Generativa

---

## 🎯 Objetivos

- Comparar respostas entre modelos
- Estruturar prompts em XML
- Melhorar consistência de saída
- Testar performance contextual

---

## 🧠 Conceitos Trabalhados

- XML Prompt Engineering
- Estrutura hierárquica de prompts
- Controle contextual
- Comparação de modelos de IA
